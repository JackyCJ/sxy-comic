# sxy-comic
comic downloader tool

[点击下载](https://github.com/JackyCJ/sxy-comic/releases/download/v0.1/comic_downloader_v0.1.exe)

使用步骤：

1. 搜索关键字。

2. 选中搜索结果。

3. 选择下载章节，下载。

4. 等待下载完成。
